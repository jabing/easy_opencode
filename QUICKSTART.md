# Quick Start: Easy OpenCode

## 1) Prerequisites

```bash
brew install opencode
# or
npm install -g opencode
opencode --version
```

## 2) Install Plugin

```bash
npm install -g easy-opencode
```

## 3) Install into a Project

```bash
cd your-project
eoc-install --project --yes
```

Or install globally:

```bash
eoc-install --global --yes
```

## 4) Verify in OpenCode

```bash
opencode
```

Inside OpenCode run:

- `/agents`
- `/help`

- 14 agents

---
description: Export platform API views for UI/API integration.
argument-hint: <report|runs|release|exporters|ui-overview> [--json]
---

Export productized platform views over runs, release decisions, telemetry, and artifacts.

Examples:
- `/platform-report report --json`
- `/platform-report runs --json`
- `/platform-report release --policy production --json`
- `/platform-report ui-overview --json`
